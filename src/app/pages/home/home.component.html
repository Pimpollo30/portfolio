PROYECTOS

Procake Angular Firebase

Procake es un proyecto e-commerce desarrollado en Angular para una pastelería ficticia que se dedica a la producción y comercialización de productos relacionados con el sector pastelero.

Permite el inicio de sesión de usuarios mediante Firebase Authentication, la visualización de productos y un carrito de compras para los pedidos. La información es almacenada en Firestore Database.

Demo Video GitHub
Random User Generator Flutter Dart

Es una aplicación móvil desarrollada en Flutter que consume los datos de la API Random User Generator (gratuita y de código abierto) para generar datos de usuarios de manera aleatoria.

Video GitHub
Android Wear OS DataLayer Android Kotlin

Es un proyecto Android escrito en Kotlin que permite el envío unidireccional (del dispositivo móvil al dispositivo wear) usando la API de Data Layer.

Permite el envío de una imagen tomada con el dispositivo móvil y la información de sensores (temperatura, luz, presión y acelerómetro).
